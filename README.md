Showcase your chess.com profile as a widget in your GitHub home page and/or in your portfolio, blogs, social media etc

## How to use

### Embed in a ``*.md`` file

```md
![chess.com-profile-widget](https://chess-com-profile-widget.onrender.com/your-username)
```
or
```md
<p align="center">
  <img src="https://chess-com-profile-widget.onrender.com/your-username" alt="chess.com-profile-widget"/>
</p>
```

### Embed in a ``*.html`` file

```html
<img src="https://chess-com-profile-widget.onrender.com/your-username" alt="chess.com-profile-widget">
```

> You can also use ``iframe`` / ``div`` / ``embed`` tags if you prefer
